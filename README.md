### findRtan
## 게임구성
1. 캐릭터는 좌우로 계속 이동하며 벽에 부딪히거나 클릭시 진행 방향을 바꾼다
2. 파란 빗방울과 빨간 빗방울이 랜덤으로 생성되며 방울과 캐릭터가 접촉할 수 있도록 플레이 한다.
3. 파란빗방울은 +점수를 빨간 빗방울은 -점수를 받게 된다.
4. 제한 시간동안 플레이 하며 기록을 세우는 것이 게임의 최종 목표
